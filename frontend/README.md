# EduVillage Frontend

React application for the EduVillage learning platform.

## Features
- User Login
- Dashboard
- View Available Courses
- Enroll in Courses
- Unenroll from Courses
- View Enrolled Courses
- Course Details Page

## Tech Stack
- React
- Vite
- React Router
- Fetch API
- CSS

## Run the Project

1. Go to frontend folder
cd frontend

2. Install dependencies
npm install

3. Start server
npm run dev

App runs on:
http://localhost:5173
