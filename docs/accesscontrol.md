# Role-Based Access Control (RBAC)

## Roles
- Student
- Teacher
- Admin

## Permissions

### Student
- Register and login
- View and enroll in courses
- Access enrolled course content
- Attempt quizzes
- View progress

### Teacher
- Login as teacher
- Create and manage courses
- Upload lessons and materials
- Create quizzes and assignments
- View enrolled students' progress

### Admin
- Manage all users
- Approve or remove courses
- Monitor platform usage
- Access system reports

## Security Goal
Ensure users can only access features permitted by their role to maintain system integrity and data security.
