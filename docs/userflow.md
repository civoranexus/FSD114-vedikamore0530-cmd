## Student Flow
1. Student registers / logs in
2. Browses available courses
3. Enrolls in a selected course
4. Accesses course lessons (video, text, quizzes)
5. Progress is tracked automatically
6. Completes course and receives certificate (simulated)

## Teacher Flow
1. Teacher logs in
2. Creates a new course
3. Adds lessons and learning materials
4. Creates quizzes/assignments
5. Monitors student progress

## Admin Flow
1. Admin logs in
2. Manages users (students and teachers)
3. Reviews and manages courses
4. Monitors platform usage and reports
