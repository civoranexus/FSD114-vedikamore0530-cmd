## Project Overview
EduVillage is a full stack online learning platform being developed as part of the CivoraX Internship Program by Civora Nexus Pvt. Ltd.

The platform aims to provide a structured and accessible digital learning environment where students can learn, teachers can create and manage courses, and administrators can oversee the system.

## User Roles
- Student: Enrolls in courses, consumes learning content, and tracks progress
- Teacher: Creates courses, uploads content, and manages assessments
- Admin: Manages users, courses, and overall platform activity

## Core Features (Planned)
- Secure user authentication and role-based access control
- Course and lesson management
- Student enrollment and progress tracking
- Quizzes and assessments
- Administrative dashboard and reporting

## Technology Stack (Planned)
- Frontend: React
- Backend: Node.js with Express
- Database: MySQL / PostgreSQL
