# EduVillage Backend

Spring Boot REST API for EduVillage platform.

## Features
- User Registration
- User Login (JWT)
- Course List API
- Enroll / Unenroll Courses
- Get User Enrolled Courses
- Spring Security + JWT

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- MySQL

## Run Backend

1. Go to backend folder
cd backend

2. Run project
mvn spring-boot:run

Server runs on:
http://localhost:8090
